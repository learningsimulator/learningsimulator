**********************************************
Welcome to Learning Simulator's documentation!
**********************************************

*Learning Simulator* is a Python program used for simulating associative
learning mechanisms. It uses a simple and well-documented scripting
language to specify the properties of the mechanism and the description
of the experiment setup being simulated.
    
The documentation for *Learning Simulator* is hosted on `Read the
Docs <https://readthedocs.org/>`_.
